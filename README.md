# Trabajo Integrador Parte 1

## Descripción
La página presenta la discografía y trabajos de la artista Hayley Williams

## Contenido
Contiene secciones demostrando los diferentes proyectos de la artista, cada sección es acompañada de imágenes de los respectivos proyectos y links a páginas oficiales.
## Estilo
Se presentan secciones desplegables acompañadas de tablas para presentar los proyectos musicales de la artista, además se incluyen sus imágenes respectivas. El sitio presenta la reactividad para adaptarse al tamaño de la pantalla desde la que se visite.
## Funcionalidades
Todas las imágenes de la página pueden ser presionadas para mejorar su visibilidad.
Las secciones se pueden expandir o colapsar para reducir el tamaño del sitio y ver solo el contenido preferible.
Se incluye un botón para volver al inicio del sitio cuando se haya desplazado hacia abajo.
La tecla de escape permite cerrar todas las secciones abiertas para rapidamente limpiar el sitio.
## Tecnologías
Se aplica HTML para generar la disposición general de contenido que luego se estiliza con CSS. Para culminar se agregan características reactivas mediante JavaScript
