# Trabajo Integrador Parte 1

## Descripción
El proyecto comienza la creación de una página web sobre la artista Hayley Williams.

## Contenido
Contiene secciones demostrando los diferentes proyectos de la artista, cada sección es acompañada de imágenes de los respectivos proyectos y links a páginas oficiales.
## Estilo
Todas las secciones sobre sus álbumes se convirtieron en desplegables y se rediseñaron para tener los colores de cada uno.
El menú principal ahora presenta estilos mejorados con hover para incluir íconos de cada sección a la que hace referencia y un menú inferior desplegable para redirigir a cada album individualmente.
Además, todas las imágenes en la página pueden ser ampliadas al pasar el mouse sobre ellas.
